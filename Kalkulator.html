Uₙ

Uₙ = a + (n−1)·b

Sₙ

Sₙ = ½·n·(2a+(n-1)b)

a (Suku Pertama)

a = Uₙ − (n−1)·b

b (Beda)

b = (Uₙ − a) / (n−1)

n (Banyak Suku)

n = (Uₙ − a)/b + 1
